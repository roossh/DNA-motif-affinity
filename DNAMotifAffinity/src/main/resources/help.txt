DNA Motif Affinity Change Calculator

This program calculates the change in DNA motif affinity in a given sample file.

Files are assumed to be tab-separated VCF (Variant Call Format) files:

CHROM | ID | POS | REF | ALT | Other Columns...

Regions for the motif in question are assumed to be tab-separated BED-files:

CHROM | START | END

PFM refers to position frequency matrix, they are expected to be tab-separated.
